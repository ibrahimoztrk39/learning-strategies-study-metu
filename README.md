# A Study of Learning Strategies and Study Behaviors among METU Students

## Overview
This repository contains the final academic project report examining learning strategies and study behaviors among students at Middle East Technical University (METU). The study is based on survey data and focuses on understanding how different learning approaches relate to academic behaviors.

## Objectives
- Identify common learning strategies used by university students
- Analyze study behaviors and habits among METU students
- Examine relationships between learning strategies and academic performance

## Methodology
- Survey-based data collection
- Descriptive statistical analysis
- Interpretation of study behavior patterns

## Key Topics
- Learning strategies
- Study behaviors
- University student learning habits
- Statistical analysis
